📌 Project Overview
A job board application built using Angular with CI/CD automation using GitHub Actions and Vercel.

✨ Features
- Job listing UI
- Search and filter jobs
- Responsive design
- Clean UX
- Fast performance

⚙️ Tech Stack
- Angular
- TypeScript
- GitHub Actions (CI/CD)
- Vercel (Deployment)

🔄 CI/CD Pipeline
- GitHub Actions runs on every push
- Installs dependencies
- Builds Angular project
- Vercel automatically deploys latest version

🚀 Deployment
Hosted on Vercel with automatic deployment from GitHub main branch.
